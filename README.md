##公司简介<br>
铂渊信息技术（上海）有限公司，专注于全平台的高清实时通话云服务解决方案。AnyRTC云适用于有音视频通话需求的所有领域。

### 项目功能
1. 高清视频会议和语音通话和文本聊天。
2. 多达4人免费视频会议在同一时间在同一房间。
3. 自己创建房间简单分享，点击链接即可入会。
4. 消息实时提醒。
5. 无需注册。
